# Active Docs - SPHERE II Entrepreneurship

This folder is for active entrepreneurship research work inside K R&D Lab.

Use `docs/` for:

- venture and opportunity execution notes
- market, audience, and behavioral signal staging
- ecosystem and partnership research packs
- public case skeletons
- GPT-assisted venture research workflows

Historical or superseded material should move to:

- `archived_docs/`

## Folder structure

- `docs/core/`
  - cross-track entrepreneurship execution notes
  - E1-E4 research execution index
  - structure approval gate
  - GPT-assisted entrepreneurship workflow

## Current active logic

SPHERE II should not become generic startup content.

It should answer:

- what decision, opportunity, market, or ecosystem question is being explored
- what evidence would validate or weaken the interpretation
- what practical action follows for a founder, operator, partner, or public case
- how the work connects to `S`, `E+T`, or `S+E+T` without blurring the primary sphere

## Before execution

Use `docs/core/ENTREPRENEURSHIP_STRUCTURE_APPROVAL_GATE.md` before turning a venture topic into a research artifact.

Use `docs/core/E1_E4_RESEARCH_EXECUTION_INDEX.md` as the cross-track index for deciding which entrepreneurship route to execute next.

Use `docs/core/GPT_ASSISTED_ENTREPRENEURSHIP_WORKFLOW.md` when an entrepreneurship research pack is being sent into GPT and returned as repo-ready `.md` output.
